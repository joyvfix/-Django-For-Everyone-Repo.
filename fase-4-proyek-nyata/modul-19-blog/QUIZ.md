**Jawaban Soal 1:** Apa itu slug dan mengapa berguna untuk URL?

> ✅ Versi URL-friendly dari judul: 'Belajar Django' → 'belajar-django'. SEO-friendly dan mudah dibaca manusia.

---

**Jawaban Soal 2:** Library apa yang digunakan untuk rich text editor di Django?

> ✅ TinyMCE, CKEditor, Quill, Summernote — semua punya Django package wrapper.

---

**Jawaban Soal 3:** Apa itu RSS feed dan untuk apa digunakan?

> ✅ Format XML untuk sindikasi konten. Memungkinkan reader app (Feedly, dll) subscribe dan otomatis dapat artikel baru.

---

**Jawaban Soal 4:** Bagaimana cara implementasi sistem like tanpa reload halaman?

> ✅ Menggunakan JavaScript (fetch/AJAX) untuk kirim request ke API endpoint, update UI tanpa full page reload.
