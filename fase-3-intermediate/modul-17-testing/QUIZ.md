**Jawaban Soal 1:** Apa perbedaan unit test dan integration test?

> ✅ Unit test: test satu fungsi/method secara terisolasi. Integration test: test interaksi antar komponen (view + model + template).

---

**Jawaban Soal 2:** Apa itu test coverage?

> ✅ Persentase kode yang dieksekusi saat test berjalan. Coverage 80% berarti 80% kode sudah di-test.

---

**Jawaban Soal 3:** Perintah apa untuk menjalankan semua test Django?

> ✅ python manage.py test

---

**Jawaban Soal 4:** Apa kegunaan Django TestClient?

> ✅ Browser simulasi untuk test view — bisa kirim GET/POST request dan cek response tanpa buka browser sungguhan.

---

**Jawaban Soal 5:** Mengapa test harus independen satu sama lain?

> ✅ Test yang bergantung pada test lain tidak reliable. Urutan test bisa berubah, dan satu test gagal bisa cascade ke test lain.
