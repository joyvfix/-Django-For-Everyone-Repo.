## 🌅 Checklist Harian

- [ ] Satu fungsi = satu tugas. Kalau fungsi terlalu panjang, pecah jadi beberapa fungsi
- [ ] Beri nama function yang mendeskripsikan apa yang dilakukan: hitung_total() bukan ht()
- [ ] Tulis test sederhana untuk setiap fungsi yang kamu buat
- [ ] Review kode orang lain di GitHub — baca dan pelajari cara mereka menulis
- [ ] Commit ke Git setiap selesai satu fitur kecil — jangan tunggu sempurna

## 🧠 Growth Mindset

> _"Saya belum bisa melakukan ini **BELUM**."_

Kalau kamu stuck, itu normal. Semua developer pernah stuck.
Yang membedakan adalah: **apakah kamu berhenti atau terus mencoba?**

**Ketika stuck:**
1. ⏸️ Istirahat 10 menit — jalan-jalan, minum air
2. 🔍 Baca ulang error message dengan pelan
3. 🌐 Google dengan kata kunci spesifik
4. 📝 Tulis ulang masalahnya dengan kata-kata sederhana
5. 💬 Tanya di komunitas — tidak ada pertanyaan yang bodoh
