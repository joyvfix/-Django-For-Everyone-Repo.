**Jawaban Soal 1:** Apa perbedaan antara class dan object?

> ✅ Class adalah blueprint/cetakan. Object adalah instance (hasil) dari class. Class 'Mobil' → object mobil_saya = Mobil()

---

**Jawaban Soal 2:** Apa itu __init__ dalam Python class?

> ✅ Constructor method yang otomatis dipanggil saat object dibuat. Digunakan untuk inisialisasi atribut.

---

**Jawaban Soal 3:** Apa yang dimaksud dengan inheritance?

> ✅ Class anak mewarisi atribut dan method dari class induk. MobilListrik bisa inherit dari Mobil.

---

**Jawaban Soal 4:** Kenapa kita perlu virtual environment?

> ✅ Menghindari konflik dependencies antar proyek. Proyek A butuh Django 4.x, Proyek B butuh Django 5.x — keduanya bisa berjalan tanpa masalah.

---

**Jawaban Soal 5:** Apa bedanya positional dan keyword argument?

> ✅ Positional: urutan penting, fungsi(nilai1, nilai2). Keyword: nama penting, fungsi(param1=nilai1, param2=nilai2).
