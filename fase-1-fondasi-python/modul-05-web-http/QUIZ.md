**Jawaban Soal 1:** Apa kepanjangan HTTP dan apa fungsinya?

> ✅ HyperText Transfer Protocol. Protokol komunikasi antara browser (client) dan server di internet.

---

**Jawaban Soal 2:** Sebutkan perbedaan HTTP GET dan POST!

> ✅ GET: mengambil data, parameter di URL, bisa di-bookmark. POST: mengirim data, parameter tersembunyi, untuk form/login.

---

**Jawaban Soal 3:** Apa fungsi DNS?

> ✅ Domain Name System — menerjemahkan nama domain (google.com) ke alamat IP (142.250.x.x) yang dimengerti komputer.

---

**Jawaban Soal 4:** Apa yang dimaksud client-side dan server-side?

> ✅ Client-side: kode berjalan di browser user (HTML, CSS, JS). Server-side: kode berjalan di server (Django, Python).

---

**Jawaban Soal 5:** Apa status code HTTP yang berarti 'OK', 'Not Found', dan 'Server Error'?

> ✅ 200 OK, 404 Not Found, 500 Internal Server Error
