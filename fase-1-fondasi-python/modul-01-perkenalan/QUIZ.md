**Jawaban Soal 1:** Apa perbedaan antara Python dan Django?

> ✅ Python adalah bahasa pemrograman. Django adalah web framework yang ditulis menggunakan Python. Analogi: Python = bahan bangunan, Django = gedung siap pakai.

---

**Jawaban Soal 2:** Perintah apa yang digunakan untuk mengecek versi Python di terminal?

> ✅ python --version atau python3 --version

---

**Jawaban Soal 3:** Apa fungsi virtual environment dalam Python?

> ✅ Mengisolasi dependencies setiap proyek agar tidak bentrok. Setiap proyek bisa punya versi package yang berbeda.

---

**Jawaban Soal 4:** Sebutkan 3 text editor/IDE populer untuk Python!

> ✅ VS Code, PyCharm, Sublime Text, Vim, Neovim, Jupyter Notebook

---

**Jawaban Soal 5:** Apa itu terminal/command line?

> ✅ Antarmuka teks untuk berkomunikasi dengan komputer menggunakan perintah. Berbeda dengan GUI yang menggunakan klik mouse.
