**Jawaban Soal 1:** Tuliskan kode for loop yang mencetak angka 1 sampai 5!

---

**Jawaban Soal 2:** Apa bedanya break dan continue dalam loop?

> ✅ break: hentikan loop sepenuhnya. continue: skip ke iterasi berikutnya, lanjutkan loop.

---

**Jawaban Soal 3:** Bagaimana mengakses elemen ketiga dari list = [10, 20, 30, 40]?

> ✅ list[2] → menghasilkan 30 (index dimulai dari 0)

---

**Jawaban Soal 4:** Kapan menggunakan try/except?

> ✅ Saat ada operasi yang berpotensi gagal: baca file, koneksi database, input user, request API.

---

**Jawaban Soal 5:** Tuliskan kode untuk mencetak semua nilai genap dari 1 sampai 10!

    if i % 2 == 0:
